# CryptoPilot Website

Marketing website and landing page for CryptoPilot — privacy-first
portfolio analytics for AI.

## Stack
- Next.js (App Router, static export)
- Tailwind CSS
- TypeScript

## Development
    npm install
    npm run dev

## Build
    npm run build
    # Static output in out/, ready for Cloudflare Pages

## Deploy
Cloudflare Pages. Build command: npm run build. Output directory: out.
